Python open-source platform built for automated trading on cryptocurrency exchanges

Tmatic empowers users to implement and manage multiple trading strategies simultaneously, offering real-time monitoring, financial tracking, and bot management. Despite the fact that Tmatic is written in Python, its code is well optimised and actively uses multi-threading technologies, ensuring quick launch, and processes a large number of symbols among which option chains. Timatic's architecture also includes the ability connecting to an unlimited number of exchanges concurrently.

Software Features

  • Simultaneous connection to multiple crypto exchanges.

  • Automatic and manual placement of orders and tracking their execution.

  • Monitoring of open positions and their financial outcomes.

  • Accounting of all transactions in the database.

  • Utilizing of any number of trading strategies for different instruments with separation of financial results.

  • Bot management features allow you to create, duplicate, merge, delete bots, and track their activity.

  • Currently supports Bitmex, Bybit, and Deribit marketplaces.

  • Tested under Windows, Linux, and macOS.

  • Free use according GPLv3 licence.

Who is Tmatic for

The software is designed for traders thinking about automating their strategies, and the open-source code written in the most common programming language makes this task easier. Further plans include expanding the list of available exchanges and strengthening the services for building and monitoring bots.
